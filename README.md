# Tech Stack Recommender 🤖

DecodeLabs Industrial Training | Batch 2026 | Project 3

## About
Recommends top job roles based on your skills using
Content-Based Filtering — TF-IDF + Cosine Similarity.

## How to Run
pip install pandas scikit-learn numpy
python recommender.py

## Features
- 20 job roles dataset (raw_skills.csv)
- TF-IDF vectorization + Cosine Similarity
- Match % + Salary range + Skills to learn
- Cold Start handling
- Save results to file

## Tech Stack
Python | Scikit-learn | Pandas | NumPy | TF-IDF | Cosine Similarity
